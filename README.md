# calc_personal_budget

A personal budget worksheet built with LibreOffice Calc, created with the assistance of Claude (Anthropic).

## Overview

This workbook helps you track, analyze, and visualize your personal finances in one place. It is generated by `build_workbook.py` using Python and exported to ODS via LibreOffice headless.

## Sheet Structure

### Transactions
The primary data entry sheet. Contains 100 sample transactions for 2026, sorted by date.

| Column | Description |
|--------|-------------|
| Date | Transaction date (YYYY-MM-DD) |
| Description | Payee or memo |
| Category | Income or expense category |
| Type | `Income` or `Expense` — dropdown validated |
| Amount | Dollar amount |

- Data is wrapped in a structured **Table** (database range) named `Transactions`
- Column D has a **dropdown validation** restricting entries to `Income` or `Expense`
- **Conditional formatting** colours each row green (Income) or orange (Expense) based on column D

### Monthly Summary
A pivot-style monthly breakdown with no spreadsheet formulas — all values are pre-computed by Python.

| Column | Description |
|--------|-------------|
| Month | Jan – Dec |
| Total Income | Sum of income transactions for the month |
| Total Expenses | Sum of expense transactions for the month |
| Net Balance | Income minus Expenses (green = positive, red = negative) |

Includes a clustered bar chart comparing monthly income vs expenses.

### Category Summary
A pivot-style annual breakdown by category, split into Income and Expense sections. No spreadsheet formulas.

| Column | Description |
|--------|-------------|
| Category | Income or expense category name |
| Annual Total | Sum of all transactions in that category for the year |
| % of Total | Share of the section total |

Includes a pie chart showing the annual expense breakdown by category.

## Requirements

- Python 3.12+ with packages from `requirements.txt`
- LibreOffice (version 7.x or later) for ODS conversion and viewing

## Building

```bash
pip install -r requirements.txt
python build_workbook.py
```

This generates `personal_budget.ods` in the project directory.

## Usage

1. Open `personal_budget.ods` in LibreOffice Calc
2. Add or edit transactions in the **Transactions** sheet — use the Type dropdown to classify each row
3. View monthly trends in **Monthly Summary**
4. View category breakdowns in **Category Summary**
